Shadowsocks软件介绍
SS，全称为 Shadowsocks（下文简称为SS），是一种基于 Socks5 代理方式的网络数据加密传输包的工具，可以使用其于科学上网 也就是翻墙 ，是目前最稳定，最常用的科学上网工具之一，全平台支持

SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

Shadowrocket 使用全指南
一、Shadowrocket 在 iPhone X 上的使用
在 iPhone X 上使用 Shadowrocket 的体验良好。用户可以通过以下步骤进行设置：

下载与安装：在 App Store 中搜索并购买 Shadowrocket。
配置代理节点：
打开应用，进入“配置”界面。
添加代理节点，输入相关信息（地址、端口、密码等）。
选择模式：根据需求选择全局模式或分流模式。
连接：点击连接按钮，完成后可以开始安全上网。
二、倍率设置
在 Shadowrocket 中，倍率设置通常与流量使用和速度有关。用户可以根据需求调整：

流量模式：在应用内设置，可以根据需要选择更高的倍率，以优化特定应用的网络使用。
数据监控：使用流量统计功能监控使用情况，避免超出流量限制。
三、与 WhatsApp 的兼容性
使用 Shadowrocket 时，WhatsApp 的访问通常不会受到影响。然而，用户在使用过程中可能会遇到以下情况：

连接问题：如果 WhatsApp 无法连接，尝试更换节点或调整 DNS 设置。
使用 VPN：确保 Shadowrocket 正在运行，并连接到一个稳定的节点。
通过适当的配置，用户可以流畅使用 WhatsApp。

四、Shadowrocket 协议
Shadowrocket 支持多种代理协议，用户可以根据需要选择：

Shadowsocks：最常用的协议，速度快，适合大多数用户。
Vmess：适合需要更高安全性的用户，通常用于 V2Ray 配置。
Trojan：以伪装流量的方式提升隐私，适合注重匿名性的用户。
不同协议的选择可以影响上网速度和安全性，用户可以根据自己的需求进行选择。

五、在 Switch 上的使用
虽然 Shadowrocket 主要为 iOS 设计，但也可以通过路由器配置实现 Switch 的代理功能：

路由器设置：在路由器上配置 Shadowrocket 代理，使连接到该路由器的设备（包括 Switch）都使用相同的代理设置。
使用 VPN：部分路由器支持 VPN 功能，可以尝试使用其他 VPN 方案在 Switch 上实现代理。
通过这些方法，用户可以在 Switch 上访问被限制的内容。

六、搭建节点
搭建自己的代理节点可以提高安全性和稳定性，基本步骤如下：

选择云服务提供商：推荐使用 Vultr、DigitalOcean、Linode 等。
创建服务器：在所选平台上创建一台 VPS（虚拟专用服务器）。
安装代理软件：
对于 Shadowsocks，安装相应的软件（如 shadowsocks-libev）。
对于 V2Ray，按照官方文档安装 V2Ray。
配置代理：根据软件文档配置节点信息，包括端口、密码等。
获取节点信息：配置完成后，记录下服务器的 IP 地址和相关设置，以便在 Shadowrocket 中添加。
搭建自己的节点虽然需要一定的技术基础，但能提供更灵活和私密的上网体验。
